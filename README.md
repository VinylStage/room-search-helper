# 🏠 청년 버팀목 실전 계산기 (Youth Beotimmok Calculator)

청년전용 버팀목 전세자금대출을 준비하는 청년들을 위한 맞춤형 실전 자금 계산기입니다. 
보유 현금과 예상 금리를 입력하면 **2026년 최신 규정**에 맞춘 최대 입주 가능 보증금과 대출 한도, 그리고 실제 이사 프로세스별 자금 투입 일정을 직관적으로 보여줍니다.

---

## ✨ 주요 기능

* **💰 최대 보증금 및 대출 한도 역산:** 보유 현금을 바탕으로 입주 가능한 가장 비싼 보증금과 은행에서 받을 수 있는 최대 대출금을 자동 계산합니다.
* **⏱️ 실전 자금 투입 타임라인:** 가계약(D-45)부터 본계약 5% 납입, 대출 심사, 잔금일(D-Day)까지 실제 내 통장에서 언제 얼마가 빠져나가는지 프로세스별로 시각화하여 보여줍니다.
* **📊 부대비용 및 월 고정 지출:** 매월 납부해야 할 이자와 관리비, 이사 당일 발생하는 중개수수료(복비)까지 어림잡아 현실적인 비용을 계산해 줍니다.
* **🌓 다크 모드 & 반응형 지원:** 사용자의 기기 시스템 설정에 맞춘 다크/라이트 모드 자동 전환을 지원하며, 스마트폰과 PC 화면 모두에 완벽하게 대응합니다.

---

## 📌 2026년 청년 버팀목 최신 규정 반영

본 계산기는 2026년 현재 적용되고 있는 주택도시기금의 청년전용 버팀목 전세자금대출 규정을 엄격하게 반영하여 설계되었습니다.

* **최대 대출 한도:** 1억 5,000만 원 (초과분은 현금으로 충당하도록 로직 구현)
* **대상 주택 보증금 한도:** 3억 원 이하 (3억 원 초과 시 계산 불가 처리)
* **대출 비율:** 임차 보증금의 80% 이내

---

## 🛠️ 기술 스택 및 보안 (Tech Stack & Security)

* **언어:** HTML5, CSS3, Vanilla JavaScript
* **보안 및 프라이버시:** 외부 서버(Backend) 통신이 전혀 없는 100% 클라이언트 사이드(Client-side) 스크립트로 동작합니다. 사용자가 입력하는 자산 정보나 대출 계산 내역은 그 어떤 곳에도 저장되거나 전송되지 않으므로 안심하고 사용할 수 있습니다.

---

## 🚀 배포 및 사용 방법 (GitHub Pages)

본 프로젝트는 정적 웹사이트(Static Website)로 구성되어 있어 GitHub Pages를 통해 무료로 쉽게 배포하고 사용할 수 있습니다.

1. 이 레포지토리를 Fork 하거나 코드를 다운로드하여 새로운 레포지토리를 생성합니다.
2. GitHub 레포지토리의 **Settings** > **Pages** 탭으로 이동합니다.
3. Source를 `Deploy from a branch`로 선택하고, Branch를 `main` (또는 `master`)으로 지정한 후 **Save**를 누릅니다.
4. 약 1~2분 뒤, 제공되는 URL(`https://[username].github.io/[repo-name]`)을 통해 언제 어디서나 스마트폰과 PC로 접속하여 계산기를 사용할 수 있습니다.
